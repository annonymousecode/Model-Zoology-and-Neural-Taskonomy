### Code for: Neural Regression, Representational Similarity, Model Zoology & Neural Taskonomy at Scale in Rodent Visual Cortex

*manual.ipynb* contains a tutorial on the use of the code and the methods therein. All steps for reproducing the key results are contained in this tutorial, though given file size constraints, the full set of results themselves are not included.

Major packages needed:

- pytorch >= 1.4
- [pytorch-image-models](https://github.com/rwightman/pytorch-image-models)
- scikit-learn >= 0.23
